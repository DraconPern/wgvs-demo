# WGVS demo project
Express Mongo api sample project for WGVS

## Requirements
- MongoDB server
- NPM and node

## Running
copy .sample-env to .env and edit to set various options

```
npm install
npm run test
npm run start
```
- browse to http://localhost:3000/
